Listas

listas automaticas

para fazer listas sem se preucupar com numeracões, digite a tag ''ol'' e dentro dela escreva, ''li''

ex:

  1. faça isso
  2. faça aquilo
  3. faça mais isso

para escolher a aparencia da lista digite ''type'' dentro de ''ol''

em ordem alfabetica:

  1. faça isso
  2. faça aquilo
  3. faça mais isso

em algarismos romanos:

  1. faça isso
  2. faça aquilo
  3. faça mais isso

para escolher quando começar, digite ''start'' dentro de ''ol''

  1. faça isso
  2. faça aquilo
  3. faça mais isso
